EduIA-Bot

Prototipo de chatbot educativo (Dialogflow Messenger) – Tema: IA en educación y uso ético

Cómo usar

Tutor virtual de Inteligencia Artificial

Este prototipo apoya el aprendizaje de conceptos básicos de IA, ejemplos de uso en educación y prácticas rápidas con retroalimentación. Para iniciar, abre el chat (esquina inferior derecha) y escribe Hola.

Asignatura: TIC Tema: IA en educación Enfoque: Tutoría + Práctica Idioma: Español
Uso ético: EduIA-Bot orienta y guía, pero no reemplaza el trabajo del estudiante.
Evita compartir datos personales; utiliza la IA como apoyo para comprender y practicar.
Sugerencia: escribe “Menú” Prueba: “Opción 4” Responde: A / B / C

Cómo usar el prototipo

  • 1) Abre el chat (abajo a la derecha).
  • 2) Escribe Hola para iniciar.
  • 3) Elige una opción: 1, 2, 3 o 4.
  • 4) En práctica (Opción 4), responde con A/B/C para ver retroalimentación.

Si el bot no entiende, escribe Menú para volver a las opciones.

Qué puede hacer

  • Explicar conceptos básicos de Inteligencia Artificial.
  • Mostrar ejemplos reales de IA aplicada a educación.
  • Orientar sobre el uso ético y responsable de IA.
  • Realizar mini evaluaciones con retroalimentación.

Menú rápido (para probar)

  • Opción 1: Conceptos básicos de IA
  • Opción 2: Ejemplos de IA en educación
  • Opción 3: Uso ético de la IA
  • Opción 4: Práctica / Mini quiz

Buenas prácticas

  • No compartas información personal (teléfono, cédula, correos, notas con nombres).
  • Usa el bot para comprender, resumir ideas y practicar.
  • Si es una tarea, solicita estructura y ejemplos; redacta tu versión final.
